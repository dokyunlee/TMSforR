작업자 안내 메시지 관리

작업 정보와 안내 기준을 입력하면 작업 시작 전·완료 후 메시지를 만들고, 작업자 화면에서 바로 확인할 수 있습니다.

1. 작업 정보 입력제목, 보상, 지침, 작업 조건을 먼저 정리합니다.
2. 문구 확인·수정후보 문구를 비교하고 최종 메시지를 직접 다듬습니다.
3. 작업자 화면 확인배포 전 작업자 화면에서 실제 노출 방식을 점검합니다.

작업 기본 정보

필수 입력 0 / 4

필수 항목을 채우면 바로 메시지를 만들 수 있습니다. 배포 시 작업 정보는 서버에도 저장됩니다.

작업자가 목록에서 바로 이해할 수 있는 짧은 제목을 입력합니다.

판단 대상과 행동을 한 문장으로 적습니다.

작업자 화면에 그대로 노출되는 내용입니다. 체크리스트처럼 작성하면 읽기 쉽습니다.

메시지 참고 정보 작업 부담과 기여 맥락을 함께 입력하면 안내 문구가 더 자연스러워집니다.

과장된 표현보다 실제 활용처를 담으면 신뢰도가 높아집니다.
